About Me

Hi everyone! My name's Pablo.

I'm a passionate self-taught Full Stack Web Developer based in Buenos Aires, who loves solving code issues and embarking upon new projects. In 2020, I completed my bachelor's degree in Business Administration and started my journey into the world of web applications and software solutions.

Language learning is a very interesting topic for me as well. I have an advanced level of English both spoken and written and, recently, I've been studying some basic japanese. Hopefully, I'll become a polyglot in the next few years.

One of my dreams is to master all the backend technologies and become an internationally recognized top-notch programmer. For the moment, I mainly code in PHP and TypeScript together with React library but, you can check out my other skills by going forward to the next section.
